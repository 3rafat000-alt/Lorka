# Gate 1: Discovery Checklist

**Owner:** res-lead (Hiroshi Tanaka)
**Deliverable:** Customer Journey Map (Mermaid)

## Validation

- [ ] User research conducted — interviews, surveys, analytics
- [ ] Personas evidence-based — not stereotypes, grounded in data
- [ ] Customer Journey Map (Mermaid) complete — all stages mapped
- [ ] Pain/gain map per stage — what hurts, what helps
- [ ] Competitor analysis done — landscape matrix
- [ ] Web research cited and verified — sources tagged
- [ ] Quantitative evidence collected — surveys, telemetry
- [ ] Edge-case personas considered (power user, novice, accessibility)
- [ ] Journey stages ordered by priority/impact

## Evidence Required

- [ ] Mermaid CJM diagram in `projects/<PRJ>/_artifacts/journey-map.md`
- [ ] Pain/gain table with source tags
- [ ] Competitor matrix (3+ competitors minimum)
- [ ] Research sources list [verified: url] for each

## Verification

- [ ] Gatekeeper verifies: persona not stereotype, claims groundable
- [ ] Every journey stage answers: "what is the user trying to do?"
- [ ] Competitor analysis — verifiable sources, not speculation

## Sign-off

- [ ] CJM signed by res-lead: "Gate 1 PASS — proceed to Design"
- [ ] CJM frozen — no changes without re-entering Gate 1
