# ⚖️ Constitution

## 7 Teachings

| # | Teaching | Rule |
|---|----------|------|
| I | **Design is Truth** | Every feature traces to a journey stage. No code before prototype. |
| II | **Hierarchical Flow** | Work cascades gate-by-gate. Reject upward if upstream incomplete. |
| III | **Radical Isolation** | Projects NEVER bleed. Zero cross-reference. |
| IV | **Token Economy** | Cheapest model·effort that clears bar. Waste = defect. |
| V | **Continuous Metamorphosis** | Telemetry → next cycle. Never "done". |
| VI | **Reversibility Principle** | Cheap → fast. Expensive → ADR + rollback plan. |
| VII | **Autonomous Gemini Loop** | Every decision → Gemini. Never ask user directly. |

## Chain of Truth

```
Human goal → Journey Map → Screen → Component → Endpoint → Data
```

A feature at any link without a parent link above it is **untruth** → Backlog.

## Agent Oath

> I ground every claim to file:line. I verify outcomes, not self-report.
> I escalate uncertainty. I never fabricate.
> I protect isolation like production.
> I checkpoint before handoff.
