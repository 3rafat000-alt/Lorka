
## Key Principles

| Principle | Meaning |
|-----------|---------|
| **Design is Truth** | Every feature → journey stage. No code before prototype. |
| **Hierarchical Flow** | Work cascades gate-by-gate. Reject upward if upstream incomplete. |
| **Radical Isolation** | Projects NEVER bleed. Zero cross-reference. |
| **Token Economy** | Cheapest effort that clears bar. Waste = defect. |
| **Autonomous Gemini Loop** | Every decision → Gemini → execute. Never ask user directly. |

## Gate Lifecycle

```
0 Inception → 1 Discovery → 2 Design → 3 Architecture → 4 Build → 5 Quality → 6 Staging → 7 Production → 8 Observe
```

No skip. Implementer never self-grades. Outputs verified by independent gatekeeper.
